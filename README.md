AnkiTabs - Swipe through your knowledge, one tab at a time.
Flip between multiple content tabs right inside your Anki cards. No more cramming everything onto one side, organize any topic into clean, focused tabs like Symptoms, Pathophysiology, and Treatment. Each tab holds its own content, its own cloze, and its own space to think. Study smarter. One tab at a time.

How to use:

1. Download the add-on, and restart Anki
2. Click Add to create a new card
3. At the top, click the note type dropdown
4. Select AnkiTabs from the list
5. Enter the name of the disease (or topic) in the title field, then fill in each tab with your content — for example Symptoms, Pathophysiology, and Treatment
6. Use cloze deletions freely within each tab, but make sure each cloze number is unique across all tabs, meaning don't reuse the same cloze number in different tabs. Any tab left empty will automatically be hidden during studying. Meaning no clutter, just the content you need
